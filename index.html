List

A few days ago, l have learned the lists in Python.

Then, I practised 3 questions.

First of all, for a given list which has only positive integers in it, if the element in it is an odd number, then add one so that it becomes an even number. For example, if the given list is [1,2,3,4,5,6], then the returned list should be [2,2,4,4,6,6]

Using a ‘for’ loop, each element can be judged whether it is an odd number. If it is, the original number is replaced by the new even number.

Secondly, I wrote a program code to create a list by concatenating a given list which range goes from 1 to n. For example, if the given list is [‘p’,’q’] and n=5, then the output should be [‘p1’,’q1’,’p2’,’q2’,’p3’,’q3’,’p4’,’q4’,’p5’,’q5’]

Then, I realized that this program code has a practical use. If the an retailer needs n different commodities with corresponding price, the tags can be printed using this program code.

Finally, I wrote a program code which can accept two input lists and returns a new list which contains only the unique elements from both sides.

The output is like [1,2,3,4,5,6,7,8,9]. This shows how powerful the method of function ‘set’ is ! It can find the unique elements in a list and rearrange them from the smallest to the biggest one.

The feelings after attending the BPho

On November 15th, I attended the BPho Physics competition. Although the content of the test was a little bit hard, I learned so much from the exam itself and the process of preparing for it.

At the beginning, facing so many difficult questions, I didn’t know what to do. As I was an AS level student, some of the questions are the ones that I hadn’t met before. As a result, after solving some questions, I decided to separate all the questions from 2008 to 2018 into 5 categories in Section 1 and 8 categories in Section 2.


Also, I made some comments, which is about some of my thinking process, on the side of the question.

After taking part in this exam, I have learned 3 good habits:

  • Being persistent. During the learning, not everything can be achieved without any obstacles. Although, once I wanted to give up, I kept telling myself ‘try again, try again’. Luckily, I did it, and regardless of the outcome, this spirit inspired me a lot.

  • Being confident. At the beginning, I was not confident at all, because I can solve few problems. As time went by, I began to make some progress, so I became more confident. Latter, I found that the more confident you are, the better the outcome will be, vice versa.

  • Being ambitious. Before doing a task, a target should be built up, which guided me when I was hesitated. Setting up a higher goal gives me many incentives to learn.

    What is more, after doing a lot of questions, I found that Physics was quite interesting. Many things in our daily lives can be explained by Physics. For example, one day, on my way to school, a car passed by with its trumpet on. The pitch of which changed as the car moving. A week later, I knew this is called the Doppler Effect.

Physics is very interesting!
