Being a good student requires a variety of skills including time management, teamwork, keeping good focus and facing challenges, which I have learnt during my experience of taking part in the Garbage Classification Programme which aims to determine which class the rubbish belongs to through image identification and CNN algorithm.

Nearest Ethnicity

(image from:https://image.shutterstock.com/image-photo/diversity-race-ethnicity-people-concept-260nw-527742886.jpg)

People around the world are from different ethnicities, so how close is it between different ethnicities?

Today, I wrote a code to find the nearest genes among 4, which means the nearest ethnicity group and the farthest one are found in the given 4 ethnicity groups.

The main idea here is to represent each type of genes in terms of vectors with 4 dimensions, each of which stands for the proportion of corresponding genes.

First of all, the 4 types of genes of four different ethnicities are inputted using numpy array. Also, 0,1,2,3 are used to represent Eskimos, Bantu, British and Korean respectively.

Then, the ‘vectors’ are unitized, which means to divide each component by the magnitude of the ‘vector’. Hence, I define a function called ‘unitize’ here. After this, the four unitized vectors are combined to form 1 array called ‘total’.

After unitizing the vector, the distance is calculated, and this is the core step. Also, I used the idea of ‘k nearest neighbour’ mentioned in the blog numpy. The 44 array is changed into a 4 * 1 4 array and a 1 * 4 * 4 array, and then the deduction is performed by using the broadcasting rules. Therefore, both arrays becomes a 4 * 4 * 4 array. During the change from 4 * 1* 4 to 4 * 4 * 4, it means that the types of genes (A1,A2,B,O) for each ethnicity are repeated 4 times, and during the change from 1 * 4* 4 to 4 * 4 * 4, it means that the information of 4 different ethnicities is repeated 4 times, so that the deduction is to calculate the ‘distance’ between corresponding genes.

Finally, the result is printed.

In conclusion, the main point here is:

  1. Use vector to represent the percentage of type of genes
  2. Use broadcasting to calculate the gene distance
  3. Figure out the nearest and the farthest ethnicities groups.

Numpy

Recently, I have learnt something about the numpy, which is a module used to process array or matrix. The most useful and interesting one I found is the algorithm called the ‘k nearest neighbours’ , which is used to find the nearest 2 points of a given point.

This is used to to generate a 10*2 array, in which the values are in the range [0,1).

The random points are visualised below.

This code is used to find the distance between two points using the formula: ((x-x0)^2+(y-y0)^2)^0.5.

First of all, x[:,np.newaxis,:] represents a 10 * 1 * 2 array, where 10 stands for 10 points including itself, and 1*2 stands for the coordinate of the point, because (x,y) is a 1 * 2 array.

Then, x[np.newaxis,:,:] represents a 1 * 10 * 2 array. This stands for the coordinates of 10 points including the point itself.

In order to make it possible to make a deduction, the broadcasting is used here. Both of 2 arrays are broadcasted to a 10 * 10 * 2 array, meaning that each point undergoes a deduction with other 10 points including itself. This can be illustrated using the following diagram.

Here, the partition(x,n) is used to find the nth smallest value and replace these values to the front of the array, and the rest will be arranged randomly. For example, if the input is np.partition([1,3,6,2,5],2), the output will be [1,2,3,6,5]. Hence, the argpartition() is used to find the index of 2 smallest point around the given point. Note that (k+1) is used, because the nearest point to the given point is itself.

This programme is very useful in real life. For example, DiDi or Uber, which are Taxi apps. The system will find the nearest taxi around you, and inform the driver according to your location. Today, I have read and learnt some codes about the CNN (Convolutional Neural Network) used in classification problems.

The example that I read was to train a model and use it to recognise the hand-written numerals. The trained data and the test data are from the MNIST (Mixed National Institute of Standards and Technology database).

Other parts of the codes are similar to the previous one that I have read, but the most impressive one for me was the description of the CNN model.

Cov2D means 2 dimensional convolution. This is a process that a 5*5 matrix is used as a kernel and convolute with the pixel matrix. For example, the image, kernel, output and the output matrix are shown below. In doing so, some parts of the image are treated as a whole, so that the order of number of parameters needed to be trained decreased dramatically.

Although the number of multiplication before the activation function increases a bit, CNN algorithm has reduced a significant amount of training process. Take a 1D convolution as an example, the picture above shows a full connection network, while the picture below shows a 1D CNN. The number of parameters needed to be trained in the full connection network is 5 weights and 1 bias (6 in total). By contrast, the parameters needed to be trained in the 1D CNN is 3 weights and 1 bias (4 in total), which is less than that of the traditional network. Imaging that billions of data needs to be trained, the parameters to be trained is, say, 10000 times more using full connection network. In addition, for the traditional network, the multiplication before activation function is f(w1x1+w2x2+w3x3+w4x4+w5x5+b), which has 5 multiplication, while for the CNN, the multiplication before activation function is f(w1x1+w2x2+w3x3+b), f(w1x2+w2x3+w3x4+b), f(w1x3+w2x4+w3x5+b), which has 9 multiplication. The number of multiplication may affect the operating efficiency, but the number of parameter needed to be trained will affect the training efficiency.

What is more, the pooling layer is used here. The maximum pooling layer is to select a locally maximum pixel to represent this area. An example is shown below. This pooling layer will also reduce the calculation quantities, which increases the operating and training efficiency.

Finally, the Dense layers are used too, which are full connection layers.

In conclusion:

  1. CNN can reduce the number parameter needed to be trained
  2. Pooling layer can also reduce the number of calculation

Neural-Network-based Iris Flower Classification

Today, I have read and learnt some codes about neural network used in classification.

There are 3 types of Iris Flowers, which are Setosa, Versicolour and Virginica respectively. The code is used to determine the type by considering its petal length and petal width.

image from: https://www.almanac.com/sites/default/files/styles/primary_image_in_article/public/image_nodes/iris-flowers.jpg?itok=RvCxClOE

First of all, the sepal width, sepal length, petal length, petal width and iris class are read from a csv file, and 5 numpyarrays are used to store the data.

Then, in order to visualise the patterns of the input data, graphs can be drawn using matplotlib. As shown in 2 graphs, the sepal length and sepal width do not show a clear pattern and there are many overlaps between Virginica and Versicolor, so these data are ignored although the accuracy level will drop a little bit. Instead, the petal length and petal width are used as input data, which shows clear pattern.

The next step is to pre-process the data. The petal length and corresponding petal width are combined together to form a new (2*150) array which is then transposed to a (150 * 2) array using .T method, so that one line stands for the feature of one flower i.e. length and width and there are characteristics of 150 flowers inputted. To transpose an array means to swap the column and row of the array. By the way, in order to have an easier interpretation of 3 types of Iris Flowers, 0,1,2 are given to them respectively.

In addition, as usual, the dataset is divided into training set (80%) and test set (20%).

Then, in order to simplify the following model, the 0,1,2 are changed to one-hot encoding. This means: 1 –> [1,0,0] 2 –>[0,1,0] 3 –>[0,0,1] , all of which are matrix form. This can be done using built-in method of tensorflow.

To visualise the separated data sets and training sets, the matplotlib is used again. Note that, for example, train_x[train_y == 0,1] means that to find the element in the 1st column which has the index of train_y == 0. For instance, if x = np.array([a,b,c,d,e]), y = np.array([1,2,3,1,4]), when x[y==1] is inputted, the output should be array([a,d]).

After pre-processing the data, model is needed to be described and trained. Here, 3 layers are used. There are 2 inputs to the input layer, which are width and length respectively, and the output of which has 64 values. This is the input of the hidden layer, which gives 64 outputs to the output layer. Finally, the output layer gives 3 outputs (0,1,2), indicating 3 types of Iris flowers, and the values of which is called the confidence score. The higher the confidence score in one neuron, the higher the probability the type represented by the neuron will be. Furthermore, in the first two layers, the activation function ‘relu (rectified linear unit)’ is used, and in the output layer, the function ‘softmax’ is used. By the way, the Dense layers mean that every neurons are connected to each other.

model.summary() can be used to see the structure of the model. As shown in the table, there are overall 4547 parameters needed to be trained. Additionally, the loss function used is ‘mse (mean square error)’ and the optimizer used is ‘sgd (stochastic gradient descent)’, after which the model is trained for 100 times . Note that ‘batch_size=4’ means that 4 sets of data are trained together and ‘verbose=1’ , which is a default, means that the progress bar is printed. If verbose=0, the progress bar will not be printed.

The final step is to evaluate the trained model using the test sets of data.

We can see the predicted data, which has 3 columns, showing 3 confidence score. The highest score indicates the predicted results. For example, in the 0st row, 0.5292347is the highest among 3, so the predicted result is Setosa. Therefore, the index of the highest probability is taken out using the argmax() method in numpy.

Then, we can visualize the result by plotting graphs. The Ground Truth of the test sets is drawn, and the predicted value is drawn. As shown in the graph, most predictions are accurate, but there are still 2 inaccurate points shown. What is more, the predicted areas representing 3 types of Iris Flowers are drawn, so that the distribution of 3 types of flowers is more easily seen.

In conclusion, I have learnt and reviewed 4 things basically:

  1. Plot and select a suitable dataset
  2. Simplify the data before training
  3. Train the data by constructing a neural network
  4. Evaluate the model and show the final result

For example, if B is on the left upper corner of A, than the calculation should be 3 * 0 + 3 * 1 + 2 * 2 + 2 * 0 + 2 * 0 + 0 * 1 + 3 * 0 + 1 * 1 + 2 * 2 = 3 + 4 +1 +4 = 12, which is written in the left upper corner of matrix A * B.

In conclusion, I have learnt the following briefly:

  1. Images are stored as matrix.
  2. Different values in the matrix (0-255) represents the colour of R/G/B
  3. To process the image is actually to change the pixel represented by the matrix. It is an algebraic calculation
  4. 2D Convolution is calculated by multiplying corresponding elements
  5. Binary value can show the edge of an object
  6. Images have many types, such as RGB and Grayscale. Grayscale image is often used to process, and then convert it into the RGB or BGR form.
