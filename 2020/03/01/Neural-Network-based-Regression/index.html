Neural Network-based Regression

Recently, I have learnt the Neural Network-based Regression, which means that according to train set and test set of data, a prediction can be made according to the model based on neural network.

First of all, a csv. file of operation cost and corresponding profit is read to the notebook, and the cost and profit are separated.

Then, we can use matplotlib to plot the scattered graph to see the distribution of data, and after which we can separate the whole data into test data (20%)and train data (80%).

Due to the fact that the data is too big to deal with, we need to normalise the value by dividing all data by the their mean, so that the data becomes smaller, which is easier to handle and looks more clear.

The next step is to describe a neural network, which, in my opinion, is the most important and core step. We use the sequential() to construct the model, which is multiple layers packed together. Here, 3 layers are used, which are input layer, hidden layer and output layer respectively, and there is 1 input data, which is a tuple, consisting of cost and profit, followed by 16 output to the hidden layer, followed by 16 output to the output layer. Then, after calculation in the activation function, which is relu (Rectified Linear Unit) here, the result of which becomes the output.

The neural network looks like this:

By the way, the .summary()method can be used to print the brief description of the model.

The next step is to train the model, using either .train_on_batch() or .fit(), and I choose the latter because it operates faster. The loss function used is mse (Mean Square Error) and the optimizer used is SGD (Stochastic Gradient Descent). Then. we can plot a graph to visualise the model being trained.

After training the model, we can evaluate the model in terms of cost. Furthermore, the weight of each neuron can be printed, which seems very complicated.

The final step is to change the processed data to the original data, i.e. denormalization.

The final graph looks like this:

After learning the Neural Network-based Regression, I have understood the importance of machine learning more deeply. We cannot write every code to achieve complex things, so the code must enable the programme to learn and find the common features of the data in order to predict what would happen. Just like this example, a CEO can determine its cost by predicting his/her expected profit. This increases the certainty for a company.
