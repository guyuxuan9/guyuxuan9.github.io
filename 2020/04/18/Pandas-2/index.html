Pandas 2

Posted on 2020-04-18 | In Data analysis |

Today, I learnt some functions and applications of the modules Pandas through the data about the number of bicycle passing through the Fremon Bridge in Seattle in the US.

<Image from https://upload.wikimedia.org/wikipedia/commons/4/4e/Fremont_Bridge.jpg>

First of all, the data is pre-processed by simplifying the names and drop the NaN data.

Here, the .eval() method is just like data['Total'] = data['West'] + data['East'] .

To visualise the data:

As shown in the graph, the data is too much and only some of the data is needed to be analysed, so the resampling is necessary.

Here, the mean value of 30 days are chosen to be the size of a window which rolls across all the data, and the central value is used to be the label.

In comparison, if 50 days are chosen to be the size of a window, the curve will look more smooth, which will be similar to the shape of a sinusoidal wave.

If we want to see which time period has the greatest number of bicycles passing by, data for the same period of time can be grouped by.

The data.index.dayofweek gives values of 0-6, each of which is repeated for 24 times because there are 24 hours a day, and the data is collected each hour. Then, 0-6 are given names of Monday, Tuesday, Wednesday……

Finally, the number of bicycles in weekday and weekend can be shown.

np.where(a(bool),b,c) means that if a is True, b is returned, while a is False, c is returned. Therefore, for the first 5 days(0,1,2,3,4), data.index.weekday < 5 is True, it is weekday, and for the remaining 2 days, data.index.weekday < 5 is False, it is weekend.
