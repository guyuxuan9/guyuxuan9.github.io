Matplotlib 3

Posted on 2020-05-02 | In Data analysis

Today, I have learnt more methods of matplotlib by reading codes about the scores of Marathon.

<Image from https://www.badenmarathon.de/fileadmin/Badenmarathon/Bilder/Fotos/Start_Ziel/Start_2019>

It looks like this:

There are 4 columns which are age, gender, split, and final respectively, among which the ‘split’ column stands for the time taken for half distance of Marathon race, and the ‘final’ stands for the total time taken for the Marathon race.

However, the datatype of ‘split’ and ‘final’ are all ‘object’, meaning that the time is treated as a string which is not convenient for the following processing, so the convert_time(s) function is defined, aiming to convert the datatype of ‘split’ and ‘final’ from ‘object’ to ‘timedelta’.

First of all, string before ‘:’ and after ‘:’ are splitted, so that the first string is the hour, the second is the minutes, and the third is the seconds. These values are given to ‘h’(hour), ‘m’(minute), ‘s’(second) respectively, so that a Timedelta variable is generated.

Then, the time in seconds of half distance and that of total distance is added to the original DataFrame by adding two more columns called ‘split_sec’ and ‘final_sec’.

Then, the time taken for half distance and total distance can be visualised using seaborn.

‘data[‘split_sec’] / data[‘final_sec’]’ stands for the proportion of time used in the first half distance. If the output is greater than 0.5, ‘2 * data[‘split_sec’] / data[‘final_sec’]’ will be greater than 1, so ‘1 - 2*data[‘split_sec’] / data[‘final_sec’]’ will be a negative value. This means that the last half distance takes up less proportion of time, i.e. they accelerates in the last half distance and vice versa.

A graph is plotted, which shows that only a small proportion of bars are less than 0, meaning that only a small proportion of people run faster in the last half distance. If we want to know the exact number of people who accelerates in the last half distance, the followin code can be used to find it.

The output is 251. From this mask, we know that out of about 40,000 people, only 251 people run faster in the last half distance, and maybe they have a greater chance to win the Marathon race.
