Strings

Yesterday, I have learnt more methods about how to deal with strings.

Create a password

First of all, I wrote a code to help the users to create a password which satisfies the requirements:

  1. The length of the password MUST be greater than 8

  2. The password MUST be a combination of letters & numbers. Either there are only letters or there are only numbers will NOT be allowed.

  3. There must be a Capital letter involved.
    The codes are posted below:

    The flow chart is shown below.

    At the beginning, the ‘while’ loop is used, because the number of times the users will try to create the password is unknown. Then, the ‘isalpha()’ , ‘isdecimal()’ and ‘isalnum()’ methods are used.

    isalpha(): return True if the string only contains letters

    isdecimal(): return True if the string only contains numbers

    isalnum(): return True if the string only contains numbers & letters

What is more, to decide whether a Capital letter is involved in the password, the lower() method is used. This helps to change all the letters in the string into the lowercase. If there is a Capital letter involved, the changed string will be different from the original string, while if there is no Capital letters involved, the alterred string will be exactly the same as before. Note that the numbers will NOT change if the lower() method is used.
