Dictionary 2

Today, I learn more about the ‘dictionary’ in Python.

Then, I have practised the following question.

Mean & Highest mark

This can be achieved using ‘Dictionary’ and some defined functions.

In lines 1 to 4, students’ names and their corresponding marks for 4 subjects are input into the dictionary. Here, the keys are names, because they are unique for small samples of people, and another dictionary is used for gathering marks with their corresponding subjects.

In lines 5-10, ‘mean’ function is defined by myself, which aims to find the mean score for particular subjects. Multiple assignment (多重赋值) is used as shown in line 7, which means that keys in ‘score’ will be assigned to ‘k’, and values in ‘score’ will be assigned to ‘v’, which is equivalent to k = score.keys() and v = score.values(). In addition, in line 8, ‘get’ method is used, so that when the input key has no corresponding values in the ‘score’, the return value will be 0. In doing so, one obstacle will not make the whole program crack down. Then, it is just summing and averaging.

In lines 12 -19, ‘highest’ function is defined, which aims to find the highest score. Here, I used the ‘append’ method instead of a = a + list(v.get(subject,0)) due to the fact that the results are integers which cannot be transferred to a list. Rather, ‘append’ here means that in the 1st line of ‘allStudents’, the specific subject(key), say ‘Chinese’, is being looked for. Then, the corresponding value, 88, is noted and the program will move to the next line. In the 2nd line, following the same process, 96, is noted, and so on. Furthermore, the next ‘for’ loop is used to find the person who has scored the highest mark. If the value of the ‘subject’(keys) is the maximum one, then the person(k) is noted.
