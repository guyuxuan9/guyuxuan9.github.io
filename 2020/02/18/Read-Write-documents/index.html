Read & Write documents

Posted on 2020-02-18 | In Python

Recently, I have learnt how to read and write documents in Python using absolute path (绝对路径) and relative path (相对路径).

Create IELTS vocabulary test

Given that I have got an IELTS vocabulary sources, I can randomly create several numbers of tests and corresponding answers with the same contents but different order of the question, so that the cheating can be avoided. The vocabulary source is shown below.

I open the file in the C drive and read the contents. Note that the ‘encoding = ‘UTF-8’’ is used because there are some Chinese in the documents. Otherwise, the Chinese cannot be recognised. The created documents are shown below.

Then, the ‘split()’ method is used to separate the English word and its Chinese meaning, which is followed by putting all the English into a list called ‘word’ and all the Chinese into a list called ‘chinese’. Here, I used a seemingly ‘stupid’ way to achieve this. As the list ‘content’ is like one Chinese, one word, I separate them according to their index (the word has odd one, and the Chinese has the even one), and after which I put the English word and corresponding Chinese into a dictionary, so that the Chinese meaning can be found more easily, which is not restricted by the index of the elements unlike the list. This is what line 1-17 did.

Then, I created 3 tests in the folder called ‘QA’ in the current working directory (当前工作目录) in the ‘write’ form. In the ‘Header’ part, the title of the tests are created, including name, date and the unique test number. Note that the ‘\n’ is used to achieve newline. Then, a list called ‘English’ is produced, with the same contents as the list ‘word’ but different corresponding index of the word, and this is done by the ‘random.shuffle()’

The next step is to create questions. By choosing an English word (keys) at a time and finding its corresponding Chinese meaning (values) in the dictionary, the word wanted to be test can be decided. Also, another 3 wrong options needs to be found, given that the multiple choice has 4 options. This can be done by deleting the correct answer in all the answers, and then randomly choose 3 of the new list. Finally, combine the correct answer with 3 incorrect ones, the options are finished.

The final step is to write the question and answers. The quiz form 1 is shown below.

In summary, 3 steps are required to deal with a document:

  1. Open the file using ‘open()’ method or create a file using ‘os.makedirs()’
  2. Use the ‘read()’ or ‘write()’ or ‘add()’ method to read or write or add something. Note that the ‘write()’ method will delete all the previous contents and write.
  3. Close the file using ‘close()’ method.
