nRF24 Wireless Communication

Posted on 2020-12-19 |

  • It communicates through SPI (Serial Peripheral Interface) protocol.
  • Operation supply voltage: 3.3V
  • CE (Chip Enable) is an active-HIGH (高电平有效) pin. When selected the nRF24L01 will either transmit or receive, depending upon which mode it is currently in. –> decide whether transmit or receive signals
  • CSN (Chip Select Not) is an active-LOW pin and is normally kept HIGH. When this pin goes low, the nRF24L01 begins listening on its SPI port for data and processes it accordingly. –> decide whether listen to SPI port (chip does not transmit/receive signals all the time)
  • SCK (Serial Clock) accepts clock pulses provided by the SPI bus Master. –> transmit data in rising edge or falling edge
  • MOSI (Master Out Slave In) is SPI input to the nRF24L01.
  • MISO (Master In Slave Out) is SPI output from the nRF24L01.
  • IRQ is an interrupt pin that can alert the master when new data is available to process. –> does not need to be connected in this case
