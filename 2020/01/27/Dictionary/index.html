Recently, I learnt about some uses of ‘dictionary’ in Python.

Menu & Dictionary

Then, I wrote a programme to construct a menu. The code is shown below.

In the first 7 lines, the categories of snakes are printed, during the process of which the ‘center’ and ‘ljust’ methods are used to adjust the position of the text. In the following 6 lines, I construct a list called ‘menu’, which is used to include all kinds of snakes.__ A list is used due to the fact that there is an automatic mapping between the order and the corresponding snake, which makes it easier to call the element.__

Then, the time is obtained from system, followed by a ‘while’ loop.__ By the way, the ‘while’ loop is used, because it allows users to input as many times as they want. __Afterwards, ‘if’ conditions are used to decide which item the user chooses. In the last ‘if’, the ‘order’ is treated as a ‘dictionary’, the ‘keys’ of which is the order number, and the value of which is the quantity of the chosen snake. At the end of the ‘if’ condition, the quantity of the snake chosen is updated.

Finally, if the report is required to be printed, i.e. number 4 in the menu is chosen, the quantity is printed with corresponding snakes. If the user doesn’t want to have the report, i.e. number 5 is chosen, the interface will be closed. By the way, the first condition is to judge whether the number is 5, followed by 4, and then 1,2,3, otherwise, all the codes will be executed till 4 and 5, which makes the condition meaningless.
