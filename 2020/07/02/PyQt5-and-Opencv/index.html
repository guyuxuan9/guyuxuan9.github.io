PyQt5 and Opencv

Posted on 2020-07-02 | In Dialogue with Plants |

Recently, I have learnt how to use PyQt5 and Opencv to capture image from the camera and show it on the screen.

PyQt5 is a program or module used to develop GUI (Graphic User Interface). Opencv is Open Source Computer Vision Library, which is used to process images.

​

Here are some codes:

First of all, we define a class Mainwindow, and inherit some of the functions from Qmainwindow

Qmainwindow originally looks like this:

We change some of the initialisation parts.

  • Opencv is used to capture image from the webcam
  • Current time is displayed using a background thread
  • Real time camera view is displayed also using a background thread

‘< h3 >’ is the HTML style font.

  • “Ret” will obtain return value from getting the camera frame, either true of false.
  • The image is stored using a 24-bit RGB format (8-8-8)
  • “#FF0000” is the hex number, each 2 digits represents one of RGB, so FF 00 00 represents red color
  • setDefaultButton –> No, so there will be a bold rectangle around No option

  • The main window provides an interface to users
  • The size of the main window is fixed to 900 * 500

  • Before the image is displayed on the screen, that part of the main window will show “The camera is initialising”

  • The promp part of the main window
  • The size is 230 * 230, a square, and the coordinates of the square is (660,30)
  • The text in the prompt is set to be read only, because it makes no sense to change the prompt
  • .setObjectName gives each object a name, so that it can be easily called outsiede the ‘definition’.
  • This is used to show time and humidity
  • Before the time is shown on the right position, it will show “Initializing”
  • Before the humidity is read from Arduino, it will show “No data”

  • This is used to set the sampling interval
  • The initial time interval of sampling is set to be 10s at the beginning
  • QtCore.Qt.AlignLeading | QtCore.Qt.AlignLeft | QtCore.Qt.AlignVCenter means to make the text keep left in horizontal direction, and keep in the centre in the vertical direction.

  • If we want to change the capture interval, we can just change the number in the textbox, and its new value will be automatically given to the variable inverval.
  • It is also important to notice that self. means the variable is like a ‘global variable’.
